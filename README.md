# Atliq Presence Insights Dashboard

This project visualizes employee attendance, work-from-home (WFH), and sick leave (SL) trends using Power BI.

##  Dashboard Overview
![atliq_hr](https://github.com/user-attachments/assets/afc2e970-4b90-4f44-b23b-f50d09598ffb)


### Key Insights:
- **Attendance %:** 91.8%  
- **WFH %:** 10.0%  
- **Sick Leave %:** 1.1%  
- Time-series trends for attendance, WFH, and SL over months  
- Day-of-week analysis for performance and presence patterns  

##  Tools Used
- Power BI  
- Excel  
- DAX  
- Data Cleaning & Visualization  

##  Purpose
This dashboard helps HR teams monitor workforce presence, identify trends, and make data-driven decisions to improve employee productivity.
